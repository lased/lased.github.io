Социальная сеть
Проект социальной сети был разработан как дипломный проект программы бакалавриата. В основе клиентской составляющей лежит фреймворк Angular, который позволяет создавать большие и сложные по части бизнес-логики приложения. Ionic позволил запустить проект на мобильных устройствах в виде приложения. Серверную часть проекта была представлена в виде API и реализована при помощи фреймворка Express. Базой данных выступала документоориентированная СУБД MongoDB, которая использует для хранения структуру JSON.
